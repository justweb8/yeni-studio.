✦ Yeni Studio
Conecta con Supabase — base de datos profesional en la nube.
1
Ve a supabase.com, crea una cuenta y un nuevo proyecto. Guarda la contraseña de la base de datos.
2
En tu proyecto ve a SQL Editor y ejecuta el archivo supabase-setup.sql que descargaste. Esto crea las 5 tablas.
3
Ve a Project Settings → API y copia la Project URL y la anon/public key.
4
Pega los datos abajo y haz clic en Conectar.
❌ No se pudo conectar. Verifica la URL y la clave.
